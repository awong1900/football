# 一起踢·足球训练营
免费开放的致力于提高足球技术的平台

## 要求：
1. 球鞋必须AG或以上，天然场草皮情况非常好，而且晚上会洒水或者露水，穿TF或布鞋会打滑。
2. 自带足球一个，如没有买的最好买个迪卡侬的F500。 大家球一样方便练习射门，因为不同球脚感不一样。
3. 自带水，食物。补充体力。
4. 最好年满16岁或以上。
5. 无费用，不聚餐，纯互助性质。

## 训练内容安排：
1. 10分钟热身
2. 15分钟传球练习，或者安排专业球员分享技术心得（不定期）
3. 45分钟专项个人训练，训练内容自己把握，本人提供业余指导
4. 20分钟分组对抗

## 时间地点：
每周5晚上7点半到9点半     
深圳大学城北大体育场

## 报名：
Q群或者微信报名
